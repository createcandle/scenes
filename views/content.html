Timer (optional)

This scene can trigger another scene. Learn more

Timers allow you to switch from one scene to another one automatically after some prefered time delay. For example to turn devices off again that the first scene switched on. But you can do much more!

Create animations and even loops

If the next scene then has a timer that in return triggers the first one, then you have created a basic 'animation' loop. You can create endlessly repeating animations this way, with as many steps as you want.

Some examples of what you could do with this:

  • Create an automation that automatically deactivates itself after some time
  • Create festive decorative light switching patterns that loop endlessly
  • Once per day set all things back to a default state
  • Create an escape room experience in your own house
